# GenLayer Price Oracle Integration

## Overview
This is a 3rd-party integration toolkit designed for **GenLayer Intelligent Contracts**. It acts as a lightweight Price Oracle, allowing smart contracts on the GenLayer network to seamlessly fetch real-time off-chain asset prices (e.g., Crypto, fiat equivalents) using external APIs like CoinGecko.

## Why is this useful for GenLayer?
Intelligent Contracts need reliable external data to function in DeFi scenarios. This Python module provides a plug-and-play solution for GenLayer developers to integrate real-time price feeds into their contracts without building the API handling logic from scratch.
